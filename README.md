# Приложение для получения статистики по продажам WB

## Описание
Приложение позволяет получать статистику по продажам на Wildberries с индивидуальными настройками для каждого пользователя.

## Функционал
- Авторизация через Telegram
- Индивидуальные настройки аккаунта:
  - API ключ
  - Артикулы товаров
- Использование базы данных для хранения данных

## Настройка окружения
База данных должна быть поднята отдельно.

### Пример файла переменных окружения (.env.example)
SECRET_KEY=<любой секретный ключ>
DATABASE_URL=mysql+mysqlconnector://<логин>:<пароль>@<ip>:<порт>/wbanaliz
TELEGRAM_BOT_TOKEN=<токен тг бота>
TELEGRAM_BOT_NAME=<имя тг бота>
TELEGRAM_URL=<домен тг бота, настраивается в Botfather>

> **Примечание:** Скопируйте `.env.example` в `.env` и заполните актуальными значениями.

### Можно запустить через Docker
См. пример файла docker-compose.yml
